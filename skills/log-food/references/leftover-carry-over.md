# Leftover / Half-Portion Carry-Over

Use this when the user says the new food is the remaining half of a previously seen plate.

## Rule
- Treat it as the same dish continuing, not a new recipe.
- Scale the earlier estimate by the stated remaining fraction, usually 50%.
- Keep the same item identity unless the user changes the food itself.
- If the user says "second half", "other half", or "remaining half", do not ask them to restate the whole meal.

## Good confirmation wording
- `Boston Pizza Spicy Buffalo Mac & Cheese, half serving (leftover half)`
- `Same dish as earlier, 50% portion`

## When to re-estimate
- The user says the prior estimate was the wrong dish.
- The user says the portion was not half after all.
- The user adds new items that were not part of the original plate.

## Example
Earlier log: Boston Pizza Spicy Buffalo Mac & Cheese, full plate = 1320 kcal

User later says: "I had the other half"

Log today as:
- Boston Pizza Spicy Buffalo Mac & Cheese (half serving) = 660 kcal

Use the same macro ratios, just scaled to the carried-over portion.
