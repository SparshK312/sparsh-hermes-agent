# Branded / packaged food lookup notes

Use this when a meal includes a branded packaged item or chain product with an official nutrition page.

## Rule
- Prefer the *official product nutrition page* over generic USDA matches when the brand/product is identifiable.
- Use the page's serving size and scale exactly to the user's stated quantity.
- If the user later corrects the portion, recompute from the same exact serving macros rather than switching to a different generic entry.

## Examples observed
- **Vector cereal**: use the official Vector nutrition page serving size, not a generic cereal proxy.
- **Lay's chips**: use the 28g / 15-chip serving from Lay's official SmartLabel or product page.
- **fairlife Core Power Vanilla 26g**: default "fairlife vanilla shake" to the 14 oz bottle unless the user specifies 42g / Elite / another flavor; use the official fairlife page macros (170 kcal, 26g protein, 6g carbs, 4.5g fat).
- **Black coffee**: keep brewed black coffee as a separate small item; don't roll it into the shake macros.

## Caution
- If the item is only partially visible in a photo and the brand/flavor can't be read, do not guess the exact product variant.
- For cans/bottles in mixed plates, do not infer zero-cal vs regular from color alone; ask or omit until clarified.
