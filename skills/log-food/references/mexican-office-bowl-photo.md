# Mexican office bowl photo pattern

Session note: office lunch photo showed a mixed Mexican-style bowl with rice, battered fried fish/cod, pulled pork, tortilla chips/nachos, guacamole, shredded cheese, sour cream, and salsa.

Rules reinforced:
- Treat the bowl as a multi-item meal, not a single generic entry.
- If a beverage can is visible on the table, do **not** count it unless the user explicitly says they drank it.
- Do not count shared chips/salsa/condiments unless the user confirms they ate them.
- Prefer conservative portion sizes when the user says the estimate feels slightly large but the item list is otherwise right.
- Log the meal to both the food log and the daily-note frontmatter; never leave the macro record in `## Notes` only.
