# Mexican restaurant photo logging notes

## When the user promises a photo
- If the user says they will send a picture or says not to log the restaurant meal yet, keep the meal open and wait for the photo before confirming or writing.
- Prefer one consolidated estimate over a partial text estimate when a photo is imminent.

## What to count from an enchilada plate
- Main entree: usually 3 enchiladas + sauce + cheese.
- Common sides: Mexican rice, refried beans, salad/pico/guac.
- Shared table items: chips and salsa shown in the photo should *not* be counted unless the user explicitly says they ate them.
- Drinks in clear cups: only count if the user names them or they’re obvious from context.

## Estimation posture
- Use the photo to identify components, then use menu/restaurant lookup for the main entree and generic USDA lookup for sides.
- If a component is ambiguous, ask once before logging rather than assuming the whole plate is a fixed combo.
