# Recurring food workflow

- When a food or snack shows up repeatedly, treat the corrected canonical entry as the source of truth.
- Prefer reusing an existing template over re-searching each time.
- If the user corrects a previous estimate, update the existing log and the template reference together.
- Keep the template small: name, serving size, macros, and any brand-specific notes.

## Known reusable example

- `hello-panda-chocolate`
  - 110 kcal bag / 21g
  - 1g protein / 13g carbs / 6g fat
  - Use for Hello Panda chocolate biscuits unless the user explicitly gives a different version.
